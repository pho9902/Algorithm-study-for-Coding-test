## 스터디

### 일시

2026-01-26(월) 시작으로 매주 월요일 4시 discord 통화로 진행

### 주제

아래 정해진 주제를 1주에 한 주제씩 모든 주제 2바퀴가 완료되면 스터디장이 준비한 코딩테스트 진행

1. 스택/큐
1. 그리디
1. 해시
1. 정렬
1. 동적 프로그래밍
1. 완전탐색
1. 이진탐색
1. DFS/BFS
1. 최단경로
1. 그래프

### 스터디 진행 방식

각 스터디원은 매주 정리한 아래 내용 단계를 통해 관리한다.

1. 매주 월요일 4시 discord 통화에서 전날(일요일) 공부한 내용 공유 및 예시 문제 라이브코테로 풀어보기
2. 화요일 ~ 토요일 매일 1시간 주제에 맞는 알고리즘 코딩 테스트 문제를 풀고 PR 생성 (1일 1pr 필수 문제를 못풀더라도 생성)
3. 각 스터디원은 PR이 생성되면 코드리뷰 작성(코멘트 필수) 스터디원 전원의 Approve 를 받은 후 Merge
4. 매주 일요일 다음주 주제에 대한 내용을 책 혹은 구글링(책에 없는 주제) 을 통해 공부
